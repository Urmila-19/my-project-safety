SafeHer વિષે

SafeHer એ મહિલાઓની સુરક્ષા માટે એક શક્તિશાળી એપ્લિકેશન છે. અમારો મુખ્ય ઉદ્દેશ્ય મહિલાઓને એક એવું સાધન પૂરું પાડવાનો છે જેનાથી તેઓ કટોકટીના સમયે ઝડપી મદદ મેળવી શકે. અમે માનીએ છીએ કે ટેકનોલોજીનો ઉપયોગ કરીને આપણે એક સુરક્ષિત સમાજનું નિર્માણ કરી શકીએ છીએ, જ્યાં દરેક મહિલા નિર્ભયતાથી જીવી શકે.

અમારી એપ્લિકેશન સરળતાથી ઉપયોગમાં લઈ શકાય તેવી રીતે ડિઝાઇન કરવામાં આવી છે, જેથી કોઈપણ પરિસ્થિતિમાં માત્ર એક બટન દબાવવાથી ત્વરિત એલર્ટ મોકલી શકાય. SafeHer સાથે, તમે ક્યારેય એકલા નથી.

Main Features

  • SOS Emergency Button: One-tap sends instant alert to family, friends, police.
  • Live Location Sharing: Shares real-time GPS location continuously until safe.
  • Fake Call Feature: A call simulation to escape unsafe situations.
  • Send SMS Alert: Pre-written emergency message goes to saved contacts.
  • Voice Activated SOS: If user shouts a keyword (e.g., "Help"), SOS activates.
  • Nearby Police Station/Help Centers: Shows nearest police station, hospital, NGOs.
  • Panic Alarm Sound: Loud siren to alert people nearby.
  • Evidence Recorder: Auto record audio/video when SOS is pressed.
  • Report Incident: Easy way to report harassment anonymously.
